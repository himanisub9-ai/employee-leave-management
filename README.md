Employee Leave Management System

A lightweight, web-based application to manage employee leave requests — built entirely with vanilla HTML, CSS, and JavaScript, with no backend or external dependencies required.

📌 Overview

Managing employee leave records often involves spreadsheets or paper trails that are hard to track and easy to lose. This project solves that with a simple, fast, browser-based tool that lets users add, view, and delete leave records instantly — no server, database, or installation needed.

It's designed to be lightweight enough to run entirely client-side, using the browser's localStorage to persist data between sessions.


✨ Features

➕ Add new leave entries with relevant details
🗑️ Delete existing leave records with a single click
💾 Persistent storage — data is saved locally in the browser and remains after refresh/close
⚡ Zero setup — runs directly in the browser, no installation or server required
🎨 Clean, responsive UI built with custom CSS

🚀 Getting Started

Run Locally


Clone the repository


bash   git clone https://github.com/himanisub9-ai/employee-leave-management.git


Navigate into the project folder


bash   cd employee-leave-management


Open index.html in your browser

Simply double-click the file, or
Use a tool like the VS Code "Live Server" extension for auto-reload during development





That's it — no npm install, no build step, no configuration.
